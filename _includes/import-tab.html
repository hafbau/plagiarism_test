Git History Editor is a simple tool intended to help you rewrite you git history.
Just import the result of your git log to get started. You'll be able to edit author names and emails, commit messages, even the time and date of your commits.

To import your commit history, execute the following command in your project directory, then paste the result below: 

git log -1000 --pretty=format:"%H*#%an*#%ae*#%at*#%s"|base64
cloud Import

Warning: Rewriting git history will change the commit's hashes, and your edited branch will not converge with the original branch.
You might not want to use this tool on commits already pushed, unless you work alone on the project.

To overwrite commits already pushed, you'll have to use git push --force.
